# C++ MFC based Hwa personal project

## 구현

![image](https://github.com/bukwon/Hwa_game/assets/148305892/b41a061c-c45b-466f-9f70-c4756fd25767)


### 화투 게임
- 메뉴 방식
  - 1(나): 2 (컴퓨터) 게임
  - 1(나): 1 (컴퓨터)게임
  - 중간 패섞기 없음

- 지정 방식
    - 1(나): 2 (컴퓨터) 게임
    - 중간 패섞기 있음

### 게임 방식
- 민화투

### 기능
- 민화투 기본 룰 구현
- 자동 패 섞기
- 보다 높은 점수를 위한 운영 구현
- 청단, 홍단 구현 없음

### 우승 조건
- 다량 점수 확보한 측이 우승
- 실시간 상대 점수와 나의 점수 공개
